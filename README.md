## About

I design and build **data infrastructure** and **AI-powered automation systems** — from raw ingestion to production-grade pipelines with intelligent error handling. My work spans the full data engineering stack, with a focus on applied AI: agent orchestration, LLM integration, and workflow automation.

I approach every system with three priorities: **reliability**, **scalability**, and **maintainability**. Whether it's designing ETL architecture, building AI agents with LangGraph, or running distributed Spark jobs — the goal is always clean, production-ready output.

> Production-grade ETL pipeline with ML-based anomaly detection that flags errors before they reach downstream systems — complete with a real-time monitoring dashboard and automated self-healing workflows.

**Key Features:**
- Multi-source data ingestion with schema validation and drift detection
- ML model that detects anomalies, failures, and data quality issues in real time
- Real-time monitoring dashboard with severity-level alerting
- Automated self-healing: configurable recovery logic on failure events
- Designed for observability, reliability, and production deployment
